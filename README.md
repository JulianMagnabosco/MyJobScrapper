# MyJobScrapper

Web Scrapper para busqueda y visualizacion de ofertas de empleos.

## Tecnologias
- Angular
- Scrapy (Python)
- ScraperAPI

## Ejecutar

- Frontend: `npm run start`
- Backend: `source venv/Scripts/activate && flask --app server.py run --debug -p 3000`

PD: No se incluyeron los spiders o sitios.
